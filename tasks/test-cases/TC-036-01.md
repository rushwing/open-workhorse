# Scenario

`inbox_write_v2()` 写入 type=request 消息时，payload 包含全部四个 delegation 字段，预期无 warn，envelope 不含 `delegation_incomplete`，且生成文件名符合新规范。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 创建 payload 文件，包含 `objective:`、`scope:`、`expected_output:`、`done_criteria:` 四个字段
3. 调用 `inbox_write_v2 "menglan" "request" "implement" "thread_036_1" "corr_036_1" "" "P1" "true" "<payload_file>"`
4. 检查生成的文件名和内容

# Expected

- 标准错误不含 "delegation incomplete" 字样
- 生成文件不含 `delegation_incomplete:` 字段
- 文件名格式匹配 `YYYYMMDDHHMMSS_request_pandas_to_menglan_corr_036_1.md`（无双下划线，无 ISO 前缀，无冒号）
