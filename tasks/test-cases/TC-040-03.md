# Scenario

`eval/baseline.json` 记录 baseline=100%，当前 mock git logs 使 compliance=78%（低于 baseline 10% 阈值），
脚本应 exit 1 并输出 `REGRESSION DETECTED`。

# Steps

1. 写入 `eval/baseline.json`：`{"compliance": 1.0}`（100%）
2. mock git logs 使 7/9 步通过（compliance≈77.8%，低于 90%）
3. 调用 eval:compliance
4. 检查退出码和输出内容

# Expected

- 退出码为 1
- 输出或 last-run.md 包含 `REGRESSION DETECTED`
- delta 行显示负值（如 `-22.2%`）
- Baseline 行显示 `100.0%`
