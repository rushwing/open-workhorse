# Scenario

`inbox_read_pandas()` 处理 ATM Envelope 格式（type=request, action=tc_complete）消息时，
能正确路由到对应 handler（`_handle_tc_complete`）。

# Steps

1. 在 `for-pandas/` 写入新格式消息文件：
   ```yaml
   ---
   message_id: msg_menglan_test
   type: request
   from: menglan
   to: pandas
   action: implement
   req_id: REQ-033
   status: success
   summary: TC complete
   ---
   ```
2. 调用 `inbox_read_pandas`
3. 检查 `for-menglan/` 是否有新消息（tc_complete success → 路由 implement）

# Expected

- exit code 0
- `for-menglan/` 中产生新消息（_handle_tc_complete → inbox_write to menglan）
- 消费后消息文件被删除
