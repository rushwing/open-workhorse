# Scenario

直接运行 `./scripts/harness.sh tc-review`（不传 PR 号），
应以非零 exit code 退出并打印用法提示。

# Steps

1. 运行 `bash scripts/harness.sh tc-review`（无参数）
2. 捕获 exit code 和 stderr

# Expected

- Exit code ≠ 0
- stderr 包含 "用法" 或 "tc-review <PR号>"
