# Scenario

REQ 文件正文包含 `API_KEY`，且 `.env.example` 中已有对应变量，
`detect_major_decision` 应返回 0（无触发）。

# Steps

1. 创建临时 REQ 文件，body 包含：`需要 TELEGRAM_BOT_TOKEN`
2. 确认 `.env.example` 中已有 `TELEGRAM_BOT_TOKEN=`（此 env 已存在）
3. 调用 `detect_major_decision <req_file>`
4. 断言返回值为 0
5. 断言 `tg_decision` 未被调用

# Expected

- 返回 0
- 无 Telegram 通知发送
