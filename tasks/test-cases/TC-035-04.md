# Scenario

Pandas 在 for-menglan/done/ 中有对应 request（correlation_id=X），但收到的 response 携带 Y≠X。

# Steps

1. 在 `for-menglan/done/` 预置 type=request 消息（correlation_id=X，req_id=REQ-035）
2. 在 `for-pandas/pending/` 写入 type=response（correlation_id=Y，Y≠X）
3. 调用 `inbox_read_pandas`

# Expected

- response 移到 `for-pandas/failed/`
- stdout 或 stderr 含 `correlation_id` 字样（warn 日志）
- 退出码为 0（全局不因单条消息失败而退出）
