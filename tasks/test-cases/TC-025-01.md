# Scenario

REQ 文件正文包含 `API_KEY` 关键词，且 `.env.example` 中无对应变量声明，
`detect_major_decision` 应返回 1 并调用 `tg_decision`。

# Steps

1. 创建临时 REQ 文件，body 包含：`需要 MY_API_KEY 才能调用外部服务`
2. 确保 `.env.example` 中无 `MY_API_KEY` 条目
3. Mock `tg_decision` 为打印函数
4. 调用 `detect_major_decision <req_file>`
5. 断言返回值为 1
6. 断言 mock `tg_decision` 被调用

# Expected

- `detect_major_decision` 返回 1
- `tg_decision` 被调用，消息包含 "MY_API_KEY" 或 "凭证"
