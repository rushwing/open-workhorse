# Scenario

`inbox_write_v2()` type=request 时，生成文件包含 `action` 和 `response_required` 字段。

# Steps

1. 调用 `inbox_write_v2 "menglan" "request" "implement" "thread_t1" "corr_t1" "" "P1" "true" ""`
2. 读取生成文件内容

# Expected

- 包含 `type: request`
- 包含 `action: implement`
- 包含 `response_required: true`
- 不包含 `event_type:` 字段（notification-only）
