# Scenario

当 REQ 的 `pending_bugs` 引用一个实际存在的 BUG 文件时，`check-req-coverage.sh` 应通过校验。

# Steps

1. 创建 stub 文件 `tasks/bugs/BUG-001.md`（内容可为最小合法 frontmatter）
2. 在 `tasks/features/` 创建临时 REQ 文件，frontmatter 含 `pending_bugs: [BUG-001]`，其他字段合法
3. 执行 `bash scripts/check-req-coverage.sh`
4. 检查退出码和输出

# Expected

- 退出码：0
- BUG-001 引用通过存在性检查，无相关错误输出
