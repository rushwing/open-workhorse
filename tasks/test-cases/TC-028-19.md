# Scenario

Bug 状态为 blocked 但未提供 blocked_reason，字段校验器应返回错误。

# Steps

1. 调用 `validateBugFields({status: "blocked", blocked_reason: ""})`（或 blocked_reason 字段缺失）
2. 断言返回值 `errors` 数组非空
3. 断言 `errors` 中至少有一条错误含 "blocked_reason required when status=blocked" 或等效说明

# Expected

- 返回至少 1 个校验错误
- 错误明确指向 blocked_reason 字段缺失
- status=blocked 时 blocked_reason 为必填字段
