# Scenario

`inboxFinalize(claimedPath, 'failed', errorSummary)` 将 claimed 文件移动到 `failed/` 目录，
并在文件末尾追加错误摘要。

# Steps

1. 使用 tmp 目录，写入一条消息并 claim
2. 获取 `claimedPath`
3. 调用 `inboxFinalize(claimedPath, 'failed', "处理失败：缺少必要字段")`

# Expected

- `claimed/` 目录下原文件不再存在
- `failed/` 目录下存在对应文件
- 文件内容末尾包含错误摘要文本 `"处理失败：缺少必要字段"`
- 函数不抛出异常
