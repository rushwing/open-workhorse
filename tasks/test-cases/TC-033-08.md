# Scenario

`inbox_read_pandas()` 处理 ATM Envelope 格式（type=notification, severity=action-required）消息时，
触发 `tg_notify` 告警。

# Steps

1. 在 `for-pandas/` 写入新格式 notification 消息文件：
   ```yaml
   ---
   message_id: msg_menglan_notif
   type: notification
   from: menglan
   to: pandas
   event_type: pipeline_failed
   severity: action-required
   ---
   ```
2. 调用 `inbox_read_pandas`（mock tg_notify）

# Expected

- exit code 0
- `tg_notify` 被调用，消息中包含 `pipeline_failed` 或文件名
- 消费后消息文件被删除
