# Scenario

将 `SHARED_RESOURCES_ROOT` 显式设置为 `/tmp/tc-021-03-<pid>`，
验证 `inbox_write` 将文件写入该路径下的 `inbox/for-menglan/`。

# Steps

1. `export SHARED_RESOURCES_ROOT=/tmp/tc-021-03-$$`
2. Source `scripts/pandas-heartbeat.sh`
3. 调用 `inbox_write "menglan" "implement" "REQ-021" "实现测试"`
4. 断言文件路径前缀为 `/tmp/tc-021-03-$$/inbox/for-menglan/`

# Expected

- 写出文件路径在 `$SHARED_RESOURCES_ROOT/inbox/for-menglan/` 下
- 不包含任何硬编码的机器特定绝对主目录前缀（macOS 或 Linux home 路径形式）
