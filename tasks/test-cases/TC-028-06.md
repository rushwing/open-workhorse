# Scenario

开发中发现修复不值得继续，从 in_progress 直接标记为 wont_fix。

# Steps

1. 构造 bug fixture：`status=in_progress, owner=menglan`
2. 调用 transition(in_progress → wont_fix)，并提供 `wont_fix_reason="root cause is upstream dependency, not fixable locally"`
3. 断言 `bug.status=wont_fix`
4. 断言 `bug.wont_fix_reason` 字段非空

# Expected

- 转换合法（不抛出 IllegalTransitionError）
- `bug.status=wont_fix`
- `bug.wont_fix_reason` 字段存在且非空
