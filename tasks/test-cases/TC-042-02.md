# Scenario

`inboxWrite()` 将 Envelope 写入 `{sharedResourcesRoot}/inbox/for-{to}/pending/` 目录，
写入文件的 YAML frontmatter 包含所有 ATM Envelope 规范必填字段。

# Steps

1. 在测试中使用 `tmp` 目录作为 `sharedResourcesRoot`
2. 构造一个合法的 `RequestEnvelope`（含 `message_id`, `type`, `from`, `to`, `created_at`, `thread_id`, `correlation_id`, `priority`, `action`, `response_required`, `objective`, `scope`, `expected_output`, `done_criteria`）
3. 调用 `inboxWrite(sharedResourcesRoot, "menglan", envelope)`
4. 读取 `{sharedResourcesRoot}/inbox/for-menglan/pending/` 下生成的文件

# Expected

- 函数返回写入文件的路径（字符串）
- 文件存在于 `pending/` 目录
- 文件以 `---` 开头（YAML frontmatter）
- frontmatter 包含 `message_id`、`type: request`、`from`、`to: menglan`
- frontmatter 包含 `created_at`（ISO 8601 UTC 格式）
- frontmatter 包含 `thread_id`、`correlation_id`、`priority`
- frontmatter 包含 `action`、`response_required`、`objective`、`scope`、`expected_output`、`done_criteria`
