# Scenario

修复者发现方案不可行，将 bug 从 in_progress 退回 open 重新评估。

# Steps

1. 构造 bug fixture：`status=in_progress, owner=menglan`
2. 调用 reopen 操作（transition: in_progress → open）
3. 断言 `bug.status=open`

# Expected

- 退回操作合法（不抛出 IllegalTransitionError）
- `bug.status=open`

> Note: bug-standard.md 仅记录此转换合法（用于重新评估），未定义 owner 写回行为。owner 处理方式由 TS 实现时依据标准确定，本 TC 不预设。
