# Scenario

修复者发现方案不可行，将 bug 从 in_progress 退回 open，owner 重置为 unassigned。

# Steps

1. 构造 bug fixture：`status=in_progress, owner=menglan`
2. 调用 reopen 操作（transition: in_progress → open）
3. 断言 `bug.status=open`
4. 断言 `bug.owner=unassigned`（或 owner 字段清空）

# Expected

- 退回操作合法（不抛出 IllegalTransitionError）
- `bug.status=open`
- `bug.owner=unassigned`（修复者释放所有权，等待重新认领）
