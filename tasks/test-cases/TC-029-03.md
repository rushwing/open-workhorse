# Scenario

`pending_bugs: []` 为合法值（表示无待处理 bug），应通过 `check-req-coverage.sh` 校验。

# Steps

1. 在 `tasks/features/` 创建临时 REQ 文件，frontmatter 含 `pending_bugs: []`，其他必填字段合法，`status` 为合法值（如 `draft`）
2. 执行 `bash scripts/check-req-coverage.sh`
3. 检查退出码和输出

# Expected

- 退出码：0
- 不出现任何与 `pending_bugs` 相关的错误信息
