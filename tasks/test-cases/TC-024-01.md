# Scenario

`TELEGRAM_BOT_TOKEN` 和 `TELEGRAM_CHAT_ID` 未设置时，
`tg_poll_commands` 应静默返回（不报错、不退出）。

# Steps

1. `unset TELEGRAM_BOT_TOKEN TELEGRAM_CHAT_ID`
2. Source `scripts/telegram.sh`
3. 调用 `tg_poll_commands`
4. 捕获 exit code

# Expected

- Exit code = 0（静默返回）
- 无 ERROR 输出
- 无网络调用（无 curl 报错）
