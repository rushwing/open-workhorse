# Scenario

`inbox_read_pandas()` 能正确处理文件名为新格式（`YYYYMMDDHHMMSS_type_from_to_target_corr.md`）的消息，routing 不依赖文件名，消息正常进入 done/。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录，初始化 inbox
2. 在 `for-pandas/pending/` 目录下写入一条消息，文件名采用新格式：
   `20260321000000_response_menglan_to_pandas_corr_REQ036_test.md`
3. 文件内容为合法 ATM Envelope（type=response, legacy_type=dev_complete, status=completed）
4. 调用 `inbox_read_pandas`

# Expected

- 消息被移动到 `for-pandas/done/`（not failed/）
- 不输出未知类型 warn
