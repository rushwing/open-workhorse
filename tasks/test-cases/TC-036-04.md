# Scenario

`inbox_read_pandas()` 能正确处理文件名为旧格式（`2026-03-21T00-00-00Z__type__from_to_target__corr.md`）的消息，保证 6 个月兼容窗口内旧消息不丢失。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录，初始化 inbox
2. 在 `for-pandas/pending/` 目录下写入一条消息，文件名采用旧格式：
   `2026-03-21T00-00-00Z__response__menglan_to_pandas__corr_REQ036_old.md`
3. 文件内容为合法 ATM Envelope（type=response, legacy_type=dev_complete, status=completed）
4. 调用 `inbox_read_pandas`

# Expected

- 消息被移动到 `for-pandas/done/`（not failed/）
- 旧格式文件名对 routing 无影响
