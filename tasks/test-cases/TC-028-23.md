# Scenario

Bug 状态为 blocked 且提供了 blocked_reason，但 blocked_from_status 为空，字段校验器应返回错误。

per check-bug-coverage.sh：status=blocked 时 blocked_reason 和 blocked_from_status 均为必填。

# Steps

1. 调用 `validateBugFields({status: "blocked", blocked_reason: "external_decision", blocked_from_status: ""})` （blocked_from_status 为空）
2. 断言返回值 `errors` 数组非空
3. 断言 `errors` 中至少有一条错误含 "blocked_from_status required when status=blocked" 或等效说明

# Expected

- 返回至少 1 个校验错误
- 错误明确指向 blocked_from_status 字段缺失
- status=blocked 时 blocked_from_status 为必填字段（与 blocked_reason 同级约束）
