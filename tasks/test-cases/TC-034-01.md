# Scenario

`inbox_read_pandas()` 扫描 `pending/`，原子 mv 成功，handler 成功，消息移到 `done/`。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 在 `for-pandas/pending/` 写入一条合法消息（type=review_blocked）
3. 调用 `inbox_init; inbox_read_pandas`

# Expected

- `for-pandas/done/` 下有与原文件同名的文件
- `for-pandas/pending/` 下该文件不存在
- 退出码为 0
