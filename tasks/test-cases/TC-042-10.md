# Scenario

当传入的 `RequestEnvelope` 缺少 delegation 必填字段（`objective`、`scope`、`expected_output`、`done_criteria` 中任意一个为空字符串或未提供）时，
`inboxWrite()` 写入文件并在 frontmatter 中标注 `delegation_incomplete: true`。

# Steps

1. 构造一个 `RequestEnvelope`，其中 `objective` 为空字符串（`""`），其余字段正常
2. 调用 `inboxWrite(sharedResourcesRoot, "menglan", envelope)`
3. 读取生成的文件 frontmatter

# Expected

- 文件成功写入（不抛出异常）
- frontmatter 包含 `delegation_incomplete: true`
- 对比：若所有 delegation 字段均非空，`delegation_incomplete` 为 `false` 或字段不存在
