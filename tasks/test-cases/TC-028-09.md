# Scenario

Bug 从任意状态进入 blocked（外部决策阻塞），unblock 后恢复 blocked_from_status，相关字段清空。

# Steps

1. 构造 bug fixture：`status=in_progress, owner=menglan`
2. 调用 block 操作：`blocked_reason=external_decision, blocked_from_status=in_progress`
3. 断言 `bug.status=blocked`
4. 断言 `bug.blocked_reason=external_decision`
5. 断言 `bug.blocked_from_status=in_progress`
6. 调用 unblock 操作
7. 断言 `bug.status=in_progress`（等于 blocked_from_status）
8. 断言 `bug.blocked_reason` 字段为空或不存在
9. 断言 `bug.blocked_from_status` 字段为空或不存在

# Expected

- unblock 后 status 恢复至 blocked_from_status（in_progress），而非其他固定状态
- blocked_reason 和 blocked_from_status 字段清空
