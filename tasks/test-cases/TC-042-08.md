# Scenario

`patchReqFrontmatter()` 只更新指定的 patch 字段，其余字段和 markdown body 保持不变。

# Steps

1. 创建一个临时 REQ markdown 文件，包含完整 frontmatter 和若干 markdown body 段落
2. 记录原始 `title`、`priority` 字段值和 body 内容
3. 调用 `patchReqFrontmatter(tmpFilePath, { status: "in_progress", owner: "menglan" })`
4. 重新读取文件并解析

# Expected

- `status` 字段更新为 `"in_progress"`
- `owner` 字段更新为 `"menglan"`
- `title`、`priority` 等未在 patch 中出现的字段保持原值不变
- markdown body（frontmatter 之后的所有内容）与原始内容完全相同（逐字节一致）
- 文件仍以 `---` frontmatter 块开头和结束
