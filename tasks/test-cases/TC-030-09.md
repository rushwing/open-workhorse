# Scenario

验证非法流转 blocked → done 被拒绝（§6.3）。

# Steps

1. 调用 validateTransition('blocked', 'done')；断言 valid=false
2. 构造 blocked req，调用 applyTransition(req, 'done')；断言抛出 IllegalTransitionError

# Expected

- 流转被拒绝，status 不变
