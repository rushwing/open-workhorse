# Scenario

验证 validateReqFields 正确检测字段违规：blocked 缺 blocked_reason、in_progress owner=unassigned、bug_linked 缺 blocked_from_owner。

# Steps

1. 调用 validateReqFields({status:'blocked', blocked_reason:'', blocked_from_status:'in_progress'})；断言有 blocked_reason 错误
2. 调用 validateReqFields({status:'in_progress', owner:'unassigned'})；断言有 owner 错误
3. 调用 validateReqFields({status:'blocked', blocked_reason:'bug_linked', blocked_from_status:'in_progress', blocked_from_owner:''})；断言有 blocked_from_owner 错误
4. 调用合法值（dep_not_done blocked）；断言 errors 为空
5. 调用合法值（in_progress + named owner）；断言 errors 为空

# Expected

- 各违规场景均返回对应字段的 ValidationError
- 合法场景返回空数组
