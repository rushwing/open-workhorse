# Scenario

以 `APP_COMMAND=pandas-heartbeat` 启动 `src/index.ts`，
断言 `scripts/pandas-heartbeat.sh` 被调用（mock 该脚本存在且可执行）。

# Steps

1. 在临时目录 mock `pandas-heartbeat.sh`（内容：`echo PANDAS_HEARTBEAT_CALLED; exit 0`）
2. 以 `APP_COMMAND=pandas-heartbeat` 启动 Node 进程（使用 tsx）
3. 捕获 stdout/stderr
4. 断言输出包含 `[pandas]` 或 shell 调用痕迹

# Expected

- 进程以 exit 0 退出
- 无 "Unknown command" 错误
- `normalizeCommand("pandas-heartbeat")` 返回 `"pandas-heartbeat"` 而不抛出异常
