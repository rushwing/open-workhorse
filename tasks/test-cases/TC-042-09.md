# Scenario

`inboxReadLegacy()` 能解析不含 `type` 字段的旧格式消息文件，返回 `LegacyMessage` 对象。

# Steps

1. 创建一个临时文件，内容为旧格式 YAML frontmatter（不含 `type` 字段），例如：
   ```
   ---
   message_id: msg_pandas_20260301120000_abcd
   from: pandas
   to: menglan
   action: implement
   req_id: REQ-001
   ---
   ```
2. 调用 `inboxReadLegacy(filePath)`

# Expected

- 函数返回非 null 的 `LegacyMessage` 对象
- `message_id`、`from`、`to`、`action` 字段值与文件内容一致
- 不抛出异常
- 若文件不存在或内容为空，返回 `null`
