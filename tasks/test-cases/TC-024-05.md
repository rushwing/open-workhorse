# Scenario

`tg_decision` 在 `TG_DECISION_TIMEOUT` 内未收到 callback，应调用 `tg_notify` 重新告警。

# Steps

1. 设置 `TG_DECISION_TIMEOUT=1`（1 秒超时）
2. Mock getUpdates 始终返回空结果（`{"ok":true,"result":[]}`)
3. 调用 `tg_decision "测试决策" "Yes" "No"`
4. 捕获 stderr 和 return code

# Expected

- 函数返回非零 exit code（超时失败）
- stderr 包含 "decision timed out"
- `tg_notify` 被调用（重新告警）
