# Scenario

当 agent 的 `pending/` 目录为空时，`inboxClaim()` 返回 `null`，不抛出异常。

# Steps

1. 使用 tmp 目录，调用 `inboxInit` 初始化目录结构（不写入任何消息）
2. 调用 `inboxClaim(sharedResourcesRoot, "menglan")`

# Expected

- 函数返回 `null`
- 不抛出任何异常
- `pending/` 和 `claimed/` 目录均为空
