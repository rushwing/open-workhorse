# Scenario

`docs/req-flow.html` 应存在且包含全部 9 个状态节点的关键词，确保 DAG 可视化覆盖完整状态机。

# Steps

1. 检查文件存在性：`test -f docs/req-flow.html`
2. 逐一验证以下 9 个关键词均出现在文件中：
   - `draft`
   - `review_ready`
   - `req_review`
   - `ready`
   - `test_designed`
   - `in_progress`
   - `blocked`
   - `review`
   - `done`
3. 可用命令：`grep -q '<keyword>' docs/req-flow.html`（每个关键词一次）

# Expected

- 文件存在（exit 0）
- 全部 9 个关键词均命中（每次 grep exit 0）
- 文件不依赖外部 CDN（无 `http://` 或 `https://` 外部资源引用）
