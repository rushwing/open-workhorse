# Scenario

`inbox_read_pandas()` 应继续处理 `for-pandas/` 扁平目录中的旧格式文件（过渡期兼容）。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 直接将消息写入 `for-pandas/`（扁平目录，非 pending/）
3. 调用 `inbox_init; inbox_read_pandas`

# Expected

- 消息文件被消费（删除）
- 退出码为 0
- 无错误
