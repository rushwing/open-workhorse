# Scenario

`parseReqFrontmatter()` 能解析 repo 中现有的 `tasks/features/REQ-033.md`，
返回的 `ReqFrontmatter` 对象字段类型与规范一致。

# Steps

1. 调用 `parseReqFrontmatter("tasks/features/REQ-033.md")`（路径相对于 repo 根目录，或传绝对路径）
2. 检查返回对象

# Expected

- 返回对象的 `req_id` 为字符串 `"REQ-033"`
- `status` 为合法 `ReqStatus` 枚举值（如 `"done"` 或其他合法值）
- `depends_on` 为数组（即使为空）
- `test_case_ref` 为数组
- `priority` 为 `"P0"` / `"P1"` / `"P2"` / `"P3"` 之一
- `pending_bugs` 为数组
- 不抛出异常
