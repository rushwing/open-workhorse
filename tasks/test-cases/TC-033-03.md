# Scenario

`inbox_write_v2()` type=response 时，生成文件包含 `in_reply_to` 字段。

# Steps

1. 调用 `inbox_write_v2 "pandas" "response" "" "thread_t1" "corr_t1" "msg_pandas_orig" "P2" "false" ""`
2. 读取生成文件内容

# Expected

- 包含 `type: response`
- 包含 `in_reply_to: msg_pandas_orig`
- 不包含 `action:` 字段（request-only）
- 不包含 `event_type:` 字段（notification-only）
