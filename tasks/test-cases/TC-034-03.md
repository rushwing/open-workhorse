# Scenario

handler 成功后，`done/` 中的文件内容与原 `pending/` 中的文件内容完全相同（未被截断或修改）。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 在 `for-pandas/pending/` 写入一条 ATM response 消息（type=response, legacy_type=dev_complete, status=completed）
3. Mock `tg_pr_ready` 以避免 Telegram 调用
4. 调用 `inbox_init; inbox_read_pandas`

# Expected

- `for-pandas/done/` 下有文件
- 文件内容包含原 frontmatter（message_id 字段）
- 文件以 `---` 开头
