# Scenario

`inbox_write_v2()` 写入 type=request 消息时，若未传入 payload_file（空字符串），预期发出 warn 并在 envelope 中写入 `delegation_incomplete: true`；消息文件仍写入（非阻断）。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 调用 `inbox_write_v2 "menglan" "request" "implement" "thread_036_7" "corr_036_7" "" "P1" "true" ""`（不传 payload_file）
3. 检查 stderr 和生成文件内容

# Expected

- 标准错误包含 "delegation incomplete" 字样
- 消息文件被写入（非阻断）
- 消息文件内容包含 `delegation_incomplete: true`
