# Scenario

for-menglan/done/ 中有已知 request（correlation_id=C1），但 response 本身没有
correlation_id 字段（Huahua 当前格式）。校验逻辑应跳过，不误判为失败。

# Steps

1. 在 `for-menglan/done/` 预置 request（correlation_id=C1，req_id=REQ-035）
2. 在 `for-pandas/pending/` 写入 type=response，**不含 correlation_id 字段**
   （legacy_type=review_complete，status=success）
3. 调用 `inbox_read_pandas`

# Expected

- response 移到 `for-pandas/done/`（resp_corr 为空，跳过校验）
- 退出码为 0
- 无 warn/error 含 correlation_id 字样
