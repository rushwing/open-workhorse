# Scenario

Bug 在 confirmed 阶段评估后决策不修复，直接标记为 wont_fix。

# Steps

1. 构造 bug fixture：`status=confirmed`
2. 调用 transition(confirmed → wont_fix)
3. 断言 `bug.status=wont_fix`

# Expected

- 转换合法（不抛出 IllegalTransitionError）
- `bug.status=wont_fix`

> Note: bug-standard.md 要求 wont_fix 必须有原因，但未定义结构化字段。原因记录方式（Agent Notes / manual review）由 TS 实现时依据标准确定，本 TC 不预设字段名。
