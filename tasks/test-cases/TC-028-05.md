# Scenario

Bug 在 confirmed 阶段评估后决策不修复，直接标记为 wont_fix。

# Steps

1. 构造 bug fixture：`status=confirmed`
2. 调用 transition(confirmed → wont_fix)，并提供 `wont_fix_reason="cost exceeds benefit"`
3. 断言 `bug.status=wont_fix`
4. 断言 `bug.wont_fix_reason` 字段非空

# Expected

- 转换合法（不抛出 IllegalTransitionError）
- `bug.status=wont_fix`
- `bug.wont_fix_reason` 字段存在且非空
