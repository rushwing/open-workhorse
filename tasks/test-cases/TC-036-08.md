# Scenario

`inbox_write_v2()` 写入 type=request 消息时，payload 在非 `references:` 块下（如 `metadata:` 块）含有缩进的 `type:` 字段，预期不发出 references 相关 warn（回归防护：[P2] 误报修复）。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 创建 payload 文件，包含全部 delegation 字段，并附加 `metadata:\n  type: internal`
3. 调用 `inbox_write_v2 "menglan" "request" "implement" "thread_036_8" "corr_036_8" "" "P2" "false" "<payload_file>"`
4. 检查 stderr

# Expected

- 标准错误不含 "references type" 字样
