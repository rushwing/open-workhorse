# Scenario

`generateMessageId(from)` 返回的 message_id 格式严格符合 `msg_{from}_{yyyymmddHHMMSS}_{rand4}`。

# Steps

1. 在测试中调用 `generateMessageId("pandas")`
2. 获取返回值
3. 检查返回值格式

# Expected

- 返回值以 `msg_pandas_` 开头
- 时间戳部分格式为 14 位数字 `YYYYMMDDHHMMSS`，对应当前 UTC 时间（误差 ≤ 2 秒）
- 末尾有 4 位随机十六进制字符（`[0-9a-f]{4}`）
- 整体格式正则：`/^msg_pandas_\d{14}_[0-9a-f]{4}$/`
- 连续调用两次返回不同值（rand4 随机性）
