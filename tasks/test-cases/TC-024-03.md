# Scenario

Telegram 收到 `hold` 指令时，`handle_telegram_commands` 应创建 `.pandas_hold` 文件。

# Steps

1. Mock getUpdates 返回 `{"text":"hold","chat":{"id":12345}}`
2. 确保 `.pandas_hold` 文件不存在
3. 调用 `handle_telegram_commands`
4. 断言 `.pandas_hold` 文件存在

# Expected

- `.pandas_hold` 文件被创建
- `auto_claim` 在持有该文件时跳过认领
