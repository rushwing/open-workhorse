# Scenario

`generateThreadId(reqId)` 返回的 thread_id 格式严格符合 `thread_{req_id}_{epoch}`。

# Steps

1. 在测试中调用 `generateThreadId("REQ-042")`
2. 获取返回值
3. 检查返回值格式

# Expected

- 返回值以 `thread_REQ-042_` 开头
- epoch 部分为 10 位十进制数字（Unix 时间戳秒级，对应当前时间，误差 ≤ 2 秒）
- 整体格式正则：`/^thread_REQ-042_\d{10}$/`
- 连续调用两次，req_id 相同时 thread_id 固定不变（thread_id 是任务链路 ID，不应每次随机变化）
