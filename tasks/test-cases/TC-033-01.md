# Scenario

`inbox_write_v2()` 生成的消息文件包含所有必填 ATM Envelope 字段，且格式正确。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 调用 `inbox_write_v2 "menglan" "request" "implement" "thread_t1" "corr_t1" "" "P1" "true" ""`
3. 读取 `for-menglan/` 中生成的文件内容

# Expected

- 文件以 `---` 开头（YAML frontmatter）
- 包含 `message_id:` 字段，值格式为 `msg_pandas_YYYYMMDDHHMMSS_xxxx`
- 包含 `type: request`
- 包含 `from: pandas`
- 包含 `to: menglan`
- 包含 `created_at:` 字段（ISO 8601 UTC 格式）
- 包含 `thread_id: thread_t1`
- 包含 `correlation_id: corr_t1`
- 包含 `priority: P1`
