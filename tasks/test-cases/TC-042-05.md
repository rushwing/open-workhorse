# Scenario

`inboxFinalize(claimedPath, 'done')` 将 claimed 文件移动到 `done/` 目录，
表示消息已处理完成。

# Steps

1. 使用 tmp 目录，写入一条消息并 claim
2. 获取 `claimedPath`（inboxClaim 返回的 path）
3. 调用 `inboxFinalize(claimedPath, 'done')`

# Expected

- `claimed/` 目录下原文件不再存在
- `done/` 目录下存在对应文件（文件名相同）
- 函数不抛出异常
