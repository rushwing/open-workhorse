# Scenario

`inbox_write_v2()` 应将消息写入 `for-{target}/pending/` 子目录，而非 `for-{target}/` 扁平目录。

# Steps

1. 设置 SHARED_RESOURCES_ROOT 为临时目录
2. 调用 `inbox_init` 然后 `inbox_write_v2 "huahua" "request" "tc_design" ...`
3. 读取 `for-huahua/pending/` 和 `for-huahua/` 扁平目录

# Expected

- `for-huahua/pending/` 下有 `.md` 文件
- `for-huahua/` 扁平目录下（不含子目录）无 `.md` 文件
