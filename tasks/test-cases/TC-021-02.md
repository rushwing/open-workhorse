# Scenario

预先在 `inbox/for-pandas/` 放置一个测试消息文件，调用 `inbox_read_pandas`，
断言处理完成后文件已被删除（消费语义）。

# Steps

1. 设置 `SHARED_RESOURCES_ROOT` 为临时目录
2. 创建 `$SHARED_RESOURCES_ROOT/inbox/for-pandas/2026-03-16-test-msg.md`，
   frontmatter: `type: review_blocked`, `req_id: REQ-999`, `summary: test`, `status: blocked`
3. Source `scripts/pandas-heartbeat.sh`
4. 调用 `inbox_read_pandas`
5. 断言消息文件不存在

# Expected

- 调用后 `for-pandas/` 目录中不存在之前创建的文件
- 脚本以 exit 0 返回（`review_blocked` 类型仅打印 warn，不报错）
