# P4 Link-to-Sale Validation

## Goal

社交媒体挂链成功完成第一笔 SKU 销售。

## In Scope

- 选品 + 定价（至少 1 个 SKU）
- 社交媒体内容生产与发布（Monkey Agent 参与）
- 购买链路打通（外链 / 跳转）
- 第一笔销售完成并记录

## Out of Scope

- 规模化销售（Phase 5+）
- 自动化内容流水线（Phase 5+）

## Exit Criteria

- [ ] 社交媒体挂链成功完成第一笔 SKU 销售
- [ ] 购买链路可追踪（UTM 或等效）
- [ ] 有用户付款记录

## Dependencies

- Phase 3 exit：有真实用户基础

## Notes

"第一笔销售"是信念验证，不是收入目标。证明链路可通即可，规模化是 Phase 5 的事。
