# P9 Ecosystem Integration

## Goal

Home Assistant / 小米生态接入 PoC 验证。

## In Scope

- Home Assistant 集成 PoC（openclaw Agent 控制智能家居设备）
- 小米生态接入 PoC（MiHome / 米家协议）
- 集成文档 + 示例配置

## Out of Scope

- 生产级智能家居套件（视 PoC 结果决定是否推进）

## Exit Criteria

- [ ] Home Assistant 接入 PoC 验证（至少控制 1 个设备）
- [ ] 小米生态接入 PoC 验证（至少读取 1 个设备状态）
- [ ] 集成方案文档发布

## Dependencies

- Phase 8 exit：Agent Market 平台就绪（生态集成可作为第三方包）

## Notes

P9 是生态扩展的探索阶段。PoC 成功则推进，失败则转向其他生态（Google Home / Matter）。
