# P3 Community PMF

## Goal

有 5–20 个真实活跃用户；有持续安装成功案例；有用户公开反馈。

## In Scope

- 公开发布（GitHub / 社交媒体）
- 安装文档 + 常见问题文档
- 用户反馈收集机制（GitHub Issues / 群聊）
- 首批种子用户招募与支持

## Out of Scope

- 付费功能（Phase 4+）
- 企业版（Phase 5）

## Exit Criteria

- [ ] 有 5–20 个真实活跃用户（非团队成员）
- [ ] 有持续安装成功案例（有记录）
- [ ] 有用户公开反馈（GitHub Issue / 社群发帖 / 推文）
- [ ] 安装成功率 > 80%（抽样验证）

## Dependencies

- Phase 2 exit：Pi 一键部署可用

## Notes

PMF 信号不等于大量用户，5 个真正依赖产品的用户 > 100 个一次性安装者。
