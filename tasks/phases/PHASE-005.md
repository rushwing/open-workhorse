# P5 Team Pack Commercialization

## Goal

企业版 Agent Team 套件可售；订阅式增值服务上线。

## In Scope

- Agent Team 套件打包（Lion + Pandas + Otter + Monkey 组合）
- 订阅定价方案设计
- 支付接入（基础）
- 企业客户 onboarding 文档

## Out of Scope

- Agent Market（Phase 8）
- 第三方 Agent 接入（Phase 8）

## Exit Criteria

- [ ] 企业版 Agent Team 套件可售（有产品页面 + 购买链路）
- [ ] 订阅式增值服务上线（至少 1 个付费套餐）
- [ ] 至少 1 个企业客户完成 onboarding

## Dependencies

- Phase 4 exit：销售链路验证完成

## Notes

"Team Pack" 的价值主张：一套设备 + 一套 Agent Team，开箱即用。订阅覆盖更新 + 支持。
