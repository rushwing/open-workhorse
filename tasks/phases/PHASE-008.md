# P8 Agent Market

## Goal

开放平台 Beta；至少 3 个第三方 Agent 包上架。

## In Scope

- Agent 包格式标准化（打包规范 + 安装协议）
- 开放平台 Beta（开发者可提交 Agent 包）
- 至少 3 个第三方 Agent 包审核上架
- 基础分成机制

## Out of Scope

- 生态系统整合（Phase 9）

## Exit Criteria

- [ ] 开放平台 Beta 上线
- [ ] 至少 3 个第三方 Agent 包上架并可安装
- [ ] 开发者文档发布

## Dependencies

- Phase 7 exit：订阅基础设施就绪

## Notes

Agent Market 是生态飞轮的起点。第一批 Agent 包可以是内部孵化，验证格式和安装流程。
