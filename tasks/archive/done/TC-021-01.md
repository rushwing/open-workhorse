# Scenario

调用 `inbox_write` 向 `for-huahua` 写入一条消息，断言输出文件存在且包含合法 YAML frontmatter。

# Steps

1. 设置 `SHARED_RESOURCES_ROOT` 为临时目录（`$TMPDIR/tc-021-01-$$`）
2. Source `scripts/pandas-heartbeat.sh`（仅加载函数，不执行 main）
3. 调用 `inbox_write "huahua" "tc_design" "REQ-021" "测试消息"`
4. 断言 `$SHARED_RESOURCES_ROOT/inbox/for-huahua/` 下存在 `.md` 文件
5. 断言文件包含 `type: tc_design`、`req_id: REQ-021`、`summary:` 字段

# Expected

- 写出的文件路径：`$SHARED_RESOURCES_ROOT/inbox/for-huahua/YYYY-MM-DD-pandas-tc-design-req-021.md`
- 文件首行为 `---`（YAML frontmatter 开始）
- 包含字段：`type`, `req_id`, `summary`, `status`
- `bash -n scripts/pandas-heartbeat.sh` 无语法错误
