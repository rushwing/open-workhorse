# Scenario

尝试将 bug 从 open 直接跳转到 closed，状态机应拒绝此非法流转。

# Steps

1. 构造 bug fixture：`status=open`
2. 调用 `validateTransition({from: "open", to: "closed"})`
3. 断言返回错误或抛出异常

# Expected

- 返回 `{valid: false}` 或抛出 `IllegalTransitionError`
- 错误信息说明 open → closed 为非法路径
- bug.status 不变，仍为 open
