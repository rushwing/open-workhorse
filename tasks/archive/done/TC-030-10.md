# Scenario

验证 tc_policy=required 时 ready → in_progress 被拒绝（§6.3：必须经过 test_designed）。

# Steps

1. 构造 req fixture：`status=ready, tc_policy=required`
2. 调用 applyTransition(req, 'in_progress')；断言抛出 IllegalTransitionError，消息含 tc_policy=required

# Expected

- 转换被拒绝，status 保持 ready
