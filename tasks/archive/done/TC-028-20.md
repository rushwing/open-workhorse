# Scenario

Bug 状态为 in_progress 但 owner 为 unassigned，字段校验器应返回错误（进行中的工作必须有明确负责人）。

# Steps

1. 调用 `validateBugFields({status: "in_progress", owner: "unassigned"})`
2. 断言返回值 `errors` 数组非空
3. 断言 `errors` 中至少有一条错误含 "owner must not be unassigned when status=in_progress" 或等效说明

# Expected

- 返回至少 1 个校验错误
- 错误明确指向 owner=unassigned 在 in_progress 状态下非法
- status=in_progress 时 owner 必须为具体的 Agent 名称（pandas/huahua/menglan/claude_code）
