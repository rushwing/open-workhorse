# Scenario

`.pandas_hold` 文件存在时，Telegram 收到 `resume` 指令应删除该文件。

# Steps

1. 预先创建 `.pandas_hold` 文件
2. Mock getUpdates 返回 `{"text":"resume","chat":{"id":12345}}`
3. 调用 `handle_telegram_commands`
4. 断言 `.pandas_hold` 文件不存在

# Expected

- `.pandas_hold` 文件被删除
- `tg_notify` 被调用发送 resume 确认
