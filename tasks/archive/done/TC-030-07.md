# Scenario

使用非 bug_linked 原因阻塞 REQ，验证 blocked_from_owner 不被设置，unblock 仅恢复 status。

# Steps

1. 构造 req fixture：`status=test_designed, owner=huahua`
2. 调用 `block(req, 'dep_not_done')`；断言 status=blocked, blocked_from_owner 不存在
3. 调用 `unblock(req)`；断言 status=test_designed，blocked_reason 清空

# Expected

- dep_not_done 不设置 blocked_from_owner
- unblock 恢复 status，owner 保持 unassigned（block 已清空）
