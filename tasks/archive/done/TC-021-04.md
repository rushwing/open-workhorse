# Scenario

在 `SHARED_RESOURCES_ROOT` 未设置（unset）的情况下，
验证脚本使用 `$HOME/Dev/everything_openclaw/personas/shared-resources` 作为默认值。

# Steps

1. `unset SHARED_RESOURCES_ROOT`
2. Source `scripts/pandas-heartbeat.sh` 并检查 `INBOX_ROOT` 变量
3. 断言 `INBOX_ROOT` 包含 `$HOME`（展开后）而非硬编码路径

# Expected

- `INBOX_ROOT` 等于 `$HOME/Dev/everything_openclaw/personas/shared-resources/inbox`
- 不包含字面量 `~`（必须已展开）
- `bash -n scripts/pandas-heartbeat.sh` 无语法错误
