# Scenario

review → blocked 时 review_round 递增；达到 2 时调用 tgNotify 通知 Daniel。

# Steps

1. 构造 req fixture：`status=review, owner=claude_code, review_round=0`
2. 第一次 `review → blocked`；断言 review_round=1，未触发 tgNotify
3. 恢复 status=review，再次 `review → blocked`；断言 review_round=2，tgNotify 被调用

# Expected

- review_round 在每次打回时递增
- review_round>=2 时 tgNotify 被调用至少一次
