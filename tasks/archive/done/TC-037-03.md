# Scenario

`feat/REQ-037` ブランチがローカルに存在しない状態で `harness.sh implement REQ-037`
を実行したとき、ブランチが自動作成されて worktree にバインドされること。

# Steps

1. 事前準備：`git branch -D feat/REQ-037` を実行してブランチを削除（存在する場合）
2. `git branch -a | grep feat/REQ-037` でブランチが存在しないことを確認
3. `./scripts/harness.sh implement REQ-037` を実行
4. `git branch --list feat/REQ-037` でブランチが作成されたことを確認
5. `git -C ~/workspace-menglan/open-workhorse branch --show-current` でブランチ確認

# Expected

- ブランチ `feat/REQ-037` が存在すること
- `~/workspace-menglan/open-workhorse/` の HEAD が `feat/REQ-037` を指していること
- スクリプトが exit 0 で終了すること
