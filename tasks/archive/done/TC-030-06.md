# Scenario

使用 block() API 以 bug_linked 原因阻塞 REQ，验证 blocked_from_owner 保存，unblock 后恢复。

# Steps

1. 构造 req fixture：`status=in_progress, owner=claude_code`
2. 调用 `block(req, 'bug_linked')`；断言 status=blocked, blocked_reason=bug_linked, blocked_from_owner=claude_code, owner=unassigned
3. 调用 `unblock(req)`；断言 status=in_progress, owner=claude_code，blocked 字段清空

# Expected

- block(bug_linked) 保存 blocked_from_owner
- unblock 从 blocked_from_owner 恢复 owner
- 所有 blocked_* 字段清空
