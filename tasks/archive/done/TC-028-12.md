# Scenario

user_bug 处于 regressing，regressing_notified=true，通知时间超过 14 天，同步触发自动关闭（含超时说明评论）。

# Steps

1. 构造 bug fixture：`bug_type=user_bug, status=regressing, github_issue=42, regressing_notified=true, regressing_notified_at=TODAY-15`
2. Mock `gh.issueView(42)` 返回 `{state: "OPEN"}`（用户未手动关闭）
3. Mock `gh.issueComment` 和 `gh.issueClose` 函数
4. 运行 GitHub 同步流程
5. 断言 `gh.issueComment` 被调用，评论内容含 "14天无响应" 或等效超时说明
6. 断言 `gh.issueClose` 被调用（自动关闭 GitHub issue）
7. 断言 `bug.status=closed`
8. 断言 `bug.agent_notes` 含超时自动关闭说明

# Expected

- 超过 14 天未响应触发自动关闭
- GitHub issue 收到超时说明评论后被关闭
- 本地 status 更新为 closed
- Agent Notes 记录"14天无响应"原因
