# Scenario

尝试将 bug 从 blocked 直接跳转到 closed，状态机应拒绝。blocked 状态必须先 unblock 才能继续流转。

# Steps

1. 构造 bug fixture：`status=blocked, blocked_reason=external_decision, blocked_from_status=in_progress`
2. 调用 `validateTransition({from: "blocked", to: "closed"})`
3. 断言返回错误或抛出异常

# Expected

- 返回 `{valid: false}` 或抛出 `IllegalTransitionError`
- 错误信息说明 blocked 必须先 unblock 才能流转到其他状态
- bug.status 不变，仍为 blocked
