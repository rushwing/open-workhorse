# Scenario

`archive_merged_reqs()` 检测到 `status: review` 的 REQ 对应 PR 为 OPEN 状态，不执行任何归档操作。

# Steps

1. 在 tmpdir 创建：
   - `tasks/features/REQ-099.md`（status: review，pr_number: 42，test_case_ref: [TC-099-01]）
   - `tasks/test-cases/TC-099-01.md`（status: ready）
2. 注入 mock `gh`：`gh pr view 42 --json state` 返回 `{"state":"OPEN"}`
3. 注入 mock `git`：记录调用（期望不被调用）
4. 调用 `archive_merged_reqs()`

# Expected

- `tasks/features/REQ-099.md` 仍存在，status 未变（仍为 review）
- `tasks/test-cases/TC-099-01.md` 仍存在，status 未变
- `tasks/archive/done/REQ-099.md` 不存在
- git commit 未被调用
