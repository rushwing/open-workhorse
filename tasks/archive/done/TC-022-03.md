# Scenario

`tasks/features/` 中同时存在 P2 和 P1 状态为 `test_designed`、`owner=unassigned` 的任务，
`auto_claim` 应选择 P1 任务。

# Steps

1. 在临时 tasks/ 目录创建：
   - `REQ-901.md`（P2, test_designed, unassigned, no deps, tc_policy=required）
   - `REQ-902.md`（P1, test_designed, unassigned, no deps, tc_policy=required）
2. 设置 `PANDAS_TASKS_DIR` 或修改 REPO_ROOT 指向临时目录
3. 调用 `auto_claim`（禁用 Telegram、禁用 detect_major_decision）
4. 断言 REQ-902 的 `owner` 字段变为 `claude_code`

# Expected

- REQ-902 被认领（owner=claude_code, status=in_progress）
- REQ-901 保持 unassigned
