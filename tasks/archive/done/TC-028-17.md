# Scenario

user_bug 处于 regressing，GitHub issue 仍为 OPEN 状态，Pandas 尝试直接推进到 closed，操作应被拒绝。

# Steps

1. 构造 bug fixture：`bug_type=user_bug, status=regressing, github_issue=42`
2. Mock `gh.issueView(42)` 返回 `{state: "OPEN"}`
3. 调用 `validateTransition({from: "regressing", to: "closed"})` 或尝试执行关闭操作
4. 断言操作被拒绝（返回错误或抛出异常）
5. 断言 `bug.status` 不变，仍为 regressing

# Expected

- 操作被拒绝；返回 `{valid: false}` 或抛出错误
- 错误信息说明 user_bug 的 closed 转换依赖 GitHub issue 已关闭
- bug.status 不变，仍为 regressing
- user_bug 的 closed 状态只能通过 GitHub 同步触发，不允许 Agent 直接推进
