# Scenario

`inbox/for-pandas/` 为空目录时，`pandas-heartbeat.sh` 应正常处理（exit 0），
无错误输出。

# Steps

1. 设置 `SHARED_RESOURCES_ROOT` 为临时目录（已创建空 `inbox/for-pandas/`）
2. 运行 `bash scripts/pandas-heartbeat.sh`
3. 捕获 exit code 和 stderr

# Expected

- Exit code = 0
- stderr 无 "ERROR" 或未捕获异常
- stdout 包含 "inbox/for-pandas/ 为空，无消息" 或类似提示
