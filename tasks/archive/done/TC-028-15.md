# Scenario

尝试将 bug 从 fixed 直接跳转到 closed，跳过必须的 regressing 阶段，状态机应拒绝。

# Steps

1. 构造 bug fixture：`status=fixed`
2. 调用 `validateTransition({from: "fixed", to: "closed"})`
3. 断言返回错误或抛出异常

# Expected

- 返回 `{valid: false}` 或抛出 `IllegalTransitionError`
- 错误信息说明 fixed 必须先经过 regressing 才能到 closed
- bug.status 不变，仍为 fixed
