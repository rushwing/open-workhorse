# Scenario

创建或更新 user_bug 时未提供 github_issue 字段，字段校验器应返回错误。

# Steps

1. 调用 `validateBugFields({bug_type: "user_bug", github_issue: ""})`（或 github_issue 字段缺失）
2. 断言返回值 `errors` 数组非空
3. 断言 `errors` 中至少有一条错误含 "github_issue required for user_bug" 或等效说明

# Expected

- 返回至少 1 个校验错误
- 错误明确指向 github_issue 字段缺失
- user_bug 必须关联 GitHub issue，此为硬性约束
