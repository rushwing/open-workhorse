# Goal

通过 LLM ReAct 循环（Claude Code CLI 或其他 LLM via SSH）在 Pi 上克隆 open-workhorse 仓库并完成 Node.js 依赖初始化，为后续服务配置和注册做准备。

# In Scope

- LLM ReAct Agent 通过 Tailscale SSH 连接 Pi
- `git clone https://github.com/rushwing/open-workhorse.git`（通过 mihomo 代理）
- `npm install`（通过代理下载 npm 依赖）
- 验证 `node_modules/` 存在，关键依赖安装成功

# Out of Scope

- `.env` 配置（REQ-010 覆盖）
- openclaw binary 安装（REQ-010 覆盖）
- systemd service 注册（REQ-011 覆盖）
- LLM ReAct 调用方式的具体实现（在 ready 阶段与 Daniel+CodeX 确定）

# Acceptance Criteria

以下为环境无关检查，可在 CI 中验证：
- `git clone` 退出码为 0，克隆目录包含 `.git/` 子目录（非空仓库）
- `npm install` 退出码为 0
- `npm run build` 退出码为 0（TypeScript 编译通过）

以下为 Pi 专属验证，**仅在目标 Pi 通过 Tailscale SSH 执行**，不纳入 CI gate：
- 服务用户家目录下克隆路径存在且包含完整仓库文件（具体路径由部署时 `DEPLOY_ROOT` 配置决定，默认 `~/open-workhorse`）
- `node_modules/` 存在，关键依赖安装成功
