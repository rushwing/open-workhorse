# Goal

通过 mihomo 代理在 Pi 上安装 Tailscale，执行 `tailscale up` 将 Pi 注册到 Tailscale 网络，使 dev 机器可通过 Tailscale IP 远程访问 Pi，为 LLM ReAct 远程部署接管做准备。

# In Scope

- 通过代理下载并安装 Tailscale（命令行一键安装）
- 执行 `tailscale up` 并完成认证
- 验证 `tailscale status` 显示 Pi 为 online 状态
- 从 dev 机器验证可通过 Tailscale IP SSH 到 Pi

# Out of Scope

- Tailscale ACL 配置（使用默认策略）
- Tailscale exit node 或 subnet router 配置
- 其他设备的 Tailscale 配置

# Acceptance Criteria

- `tailscale status` 在 Pi 上输出 Pi 节点为 online
- dev 机器可通过 `ssh user@<tailscale-ip>` 连接到 Pi
- SSH 连接无需本地局域网（可在不同网段验证）

# Test Case Design Notes

需要覆盖的场景：
- TC: Tailscale 安装脚本执行成功
- TC: tailscale up 认证成功（节点出现在控制台）
- TC: Tailscale SSH 可达性验证

注意：L3 硬件集成测试。`tailscale up` 需要 auth key（由 Daniel 提供）。
