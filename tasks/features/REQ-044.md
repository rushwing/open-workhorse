## Agent Notes

### EXISTING_BRANCH 环境变量

`_process_message` 中有 `EXISTING_BRANCH` 判断（REQ-039 单 PR 规则）。迁移时必须保留该逻辑：若 `branch_name` 字段存在，则以 `EXISTING_BRANCH=branch_name` 调用 harness。

### Keep-alive 格式

bash 版本写入的是 `runtime/menglan_alive.ts`（一个 TypeScript 文件，内容为 export const 语句）。迁移时保持该格式，因为 `stall_detection`（REQ-043）会 import 它。
