# Goal

将 `scripts/huahua-heartbeat.sh`（518 行）迁移至 `scripts/huahua-heartbeat.ts`，保持外部行为完全一致，获得真正的单元测试覆盖。

## In Scope

### 功能模块

| 模块 | bash 函数 | TS 对应 |
|------|-----------|---------|
| Frontmatter 解析 | `_get_fm_field` | 使用 REQ-042 `parseReqFrontmatter` |
| 任务回滚 | `_rollback_task` | `rollbackTask()` |
| 写 huahua response | `_write_huahua_response` | `writeHuahuaResponse()` |
| 写 fix-review 到 menglan | `_write_fix_review_to_menglan` | `writeFixReviewToMenglan()` |
| 写 tc-review 到 menglan | `_write_tc_review_to_menglan` | `writeTcReviewToMenglan()` |
| 通知失败 | `_notify_pandas_failure` | `notifyPandasFailure()` |
| 消息处理 | `_process_message` | `processMessage()` |
| 主入口 | `main()` | `main()` |

### 外部调用

- ACP session 调用（spawn Huahua coding agent）通过可 mock 的接口封装
- inbox 操作使用 REQ-042 `inboxWrite`、`inboxClaim`、`inboxFinalize`

## Out of Scope

- 修改消息处理逻辑或状态机行为
- 删除 `huahua-heartbeat.sh`（Umbrella done 后统一删除）

## Acceptance Criteria

1. `scripts/huahua-heartbeat.ts` 存在，可通过 `node --import tsx scripts/huahua-heartbeat.ts` 执行
2. `package.json` 中 `huahua:heartbeat` npm script 指向 TS 版本
3. cron entry 更新
4. `scripts/huahua-heartbeat.sh` 文件头部添加 `# @deprecated` 注释
5. `processMessage` 有单元测试，覆盖 `req_review`、`review`、`tc_design`、`bugfix` 四条主路径
6. `writeFixReviewToMenglan` 和 `writeTcReviewToMenglan` 有单元测试：写入 menglan inbox，字段正确
7. `notifyPandasFailure` 有单元测试
8. `npm run build` 通过；`npm test` 通过；`npm run release:audit` 通过

## Test Case Design Notes

- TC-045-01：`processMessage(req_review)` — 调用 agent + 结果写入 pandas inbox
- TC-045-02：`processMessage(review)` — 调用 agent + review result 写回
- TC-045-03：`processMessage(tc_design)` — 调用 agent + writeTcReviewToMenglan
- TC-045-04：`processMessage(review)` — review APPROVED → writeHuahuaResponse(approved)
- TC-045-05：`processMessage(review)` — review BLOCKED → writeFixReviewToMenglan
- TC-045-06：`rollbackTask` — 回滚后状态正确
- TC-045-07：inbox 为空时 main() 正常退出

## Agent Notes

### ACP session 封装

bash 版本通过 `openclaw session` 或类似命令 spawn Hua Hua coding agent。迁移时封装为 `spawnHuahuaSession(prompt): Promise<string>`，测试时 mock 整个函数返回 fixture 字符串。具体调用方式参考现有 `huahua-heartbeat.sh` 中的 agent spawn 逻辑。
