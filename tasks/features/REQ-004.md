# Goal

执行一次完整的端到端 smoke 测试：用户通过 Telegram 向 Lion Agent 发问，Lion 通过 openclaw 处理后返回有意义的回复，全程有可留存的证据（日志或截图）。

# In Scope

- 设计一个简单的测试问题（如"你好，你是谁？"）
- 通过 Telegram 客户端发送给 Bot
- 等待 Lion Agent 通过 openclaw 处理并回复
- 将完整对话记录保存为文件（截图、文本日志均可）

# Out of Scope

- 回复质量的定量评估（本阶段只验证路径可通）
- 多轮对话场景
- 性能或延迟指标

# Acceptance Criteria

- 存在至少一条对话记录文件（`docs/` 或 `scripts/` 目录下，或 PR 描述中附截图）
- 对话记录显示用户消息和 Lion 的实际回复（非错误信息）
- openclaw 运行期间无 unhandled exception
