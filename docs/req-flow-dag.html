v0.4 关键新增字段 / 状态

字段 / 状态类型说明
review_ready 新状态（第 9 个） Human-to-Pipeline 交接点。Daniel 设置；Pandas 原子 commit 转为 req_review（state-as-lock 防重复认领）
pending_bugs 新 frontmatter 字段 BUG-xxx 数组，可选。非空时为 Menglan 路由信号：status=req_review + pending_bugs 非空 → fix req_bug；status=test_designed → tc_review
blocked_from_status 已有字段 进入 blocked 前的状态，unblock 时恢复用。blocked_reason=bug_linked 时由 Bug 关闭自动清空
