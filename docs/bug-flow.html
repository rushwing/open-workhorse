Bug → REQ Blocking 联动（§2.2 / §2.3）

blocking 步骤（Bug 开立时）

步骤操作
1在关联 REQ 的 Agent Notes 末尾追加 Bug 外链（人工可读）
2将 BUG-xxx 加入关联 REQ 的 pending_bugs 数组（机器路由信号）
3读取 REQ 当前 status，写入 blocked_from_status
4REQ status → blocked，blocked_reason: bug_linked
5REQ owner → unassigned

unblocking 步骤（Bug 关闭时）

步骤操作
1将 BUG-xxx 追加到关联 REQ 的 ## 关联 Bug 历史 对应 bug_type 分类（永久归档）
2从关联 REQ 的 pending_bugs 数组移除本 BUG-xxx
3在 REQ Agent Notes 中更新外链状态标注为 status: closed
4若 pending_bugs == []，REQ status 恢复为 blocked_from_status
5清空 blocked_reason 和 blocked_from_status
