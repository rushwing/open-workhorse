Bug → REQ Blocking 联动（§2.2 / §2.3）

blocking 步骤（Bug 开立时）

步骤操作
1在关联 REQ 的 Agent Notes 末尾追加 Bug 外链
2读取 REQ 当前 status，写入 blocked_from_status
3REQ status → blocked，blocked_reason: bug_linked
4将 BUG-xxx 加入 REQ 的 pending_bugs 字段（v0.4 新增）
5REQ owner → unassigned

unblocking 步骤（Bug 关闭时）

步骤操作
1检查 Agent Notes 中所有 BUG 外链 status（均需 closed/done）
2从 REQ pending_bugs 移除已关闭的 BUG-xxx（v0.4 新增）
3REQ status 恢复为 blocked_from_status 的值
4清空 blocked_reason 和 blocked_from_status
